Projeto de watch dos equipamentos Hikivision instalados no clube social do Palmeiras. 
When you need usb led you must install sudo apt-get install libmagickwand-dev
"@josephdadams/usbrelay": "^1.0.3",
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
